Portfolio Website

A personal portfolio website built to showcase my profile, skills, and projects in a clean and responsive layout. This project highlights my recent work, professional details, and contact information in one place.

Features:

Responsive Navigation Bar (Menu)

Profile section with Name, Designation

Direct links to LinkedIn and GitHub

About Me section describing my background and interests

Recent Projects section with short descriptions and direct GitHub links

Resume section with a downloadable resume (Google Drive link)

Contact Details for easy communication

Technologies Used:

React.js – Component-based UI development

Tailwind CSS – Modern, responsive styling

JavaScript – Application logic and interactions

HTML – Structure and content

Purpose:

This portfolio is designed to present my skills, projects, and experience in a professional way and to serve as a central place for recruiters and collaborators to learn more about me.

Short Version (for GitHub About section):

Personal portfolio website built using React.js and Tailwind CSS to showcase my projects, resume, and contact details with a clean and responsive UI.
